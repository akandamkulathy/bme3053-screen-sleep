# bme3053-screen-sleep

The objective of this project is to assess the relationship between screen time and sleep quality. Screen exposure’s effects have been heavily documented and hold implications as a public health concern with its adverse mental and physical health outcomes. 

The effect of screen time on whether one experiences satisfactory or poor sleep must first be established. If the effect is established, it is then feasible to train a model to use screen time to identify the risk of poor sleep quality. The project will then aim to assess the accuracy of the classification with a dataset.

The code document (bme3053c_screenTime_sleep.ipynb) includes:
- Dataset loading
- Descriptive statistics
- Selection between age and screen time as the primary split
- Model design (Decision Tree Classifier)
- Evaluation of the model
